This is the `Admin Copilot` operator surface.

It is not a generic chat box. It starts with the current settings page context already attached.

Use it for three kinds of work:

- let it inspect the current page, logs, runtime state, and selected targets by itself
- use the `ISSUE` / `PR` recipes to move directly into a delivery path
- continue into repair, verification, and closeout when needed

The key point is that it already has a full self-inspection view, so you do not need to restate all the context manually first.
