欢迎来到研究启动页

这里是所有研究项目的起点。就像游戏的主菜单，你可以从这里开始新的探索。

**第一次使用？跟着这三步走：**

1. 先看看页面上的几个主要按钮
2. 暂时不用管高级设置
3. 跟着教程走一遍，5分钟就能上手

**这页是做什么的？**

把你脑海中模糊的想法，变成一个可以真正执行和验证的研究项目。

**写研究目标时，记得说清楚三件事：**

- 我想研究什么问题
- 什么结果算成功
- 需要验证哪些条件

**好的例子：**

`复现方法 X 在数据集 Y 上的结果，与 baseline Z 比较，确认在单张 24GB GPU 上能训练完成。`

提示：别着急，教程会一步步带你熟悉每个功能。
